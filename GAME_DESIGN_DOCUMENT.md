# STAR FOX SPACE SHOOTER — 游戏设计文档

> 本文档供剧情编写参考。详细描述了游戏的视觉风格、核心机制、关卡设计、角色/敌人/Boss 的造型颜色等。

---

## 一、游戏概览

- **标题**：STAR FOX SPACE SHOOTER（星际火狐 极简版）
- **类型**：2.5D 透视空间射击 / 横版射击
- **画面风格**：纯黑背景 + 镂空线条（wireframe），无大色块填充，类似矢量/雷达屏风格
- **分辨率**：800×600
- **帧率**：锁定 60fps
- **音乐**：合成器 procedural BGM（旋律+贝斯脉冲），Boss 战切换为更紧促的变奏
- **音效**：全部为程序生成的合成音（正弦波/方波/噪声），无预录音频素材

---

## 二、主角：星际战斗机

### 第一章造型（俯视透视视角）
- **颜色**：纯白线条 (255, 255, 255)，镂空无填充
- **形状**：
  - 三角形机身：尖锐机头朝上，宽大尾翼在后
  - 左右两侧各有一片后掠式三角翼（delta wing）
  - 尾部有一条竖直尾翼延伸线（从机身尾部向后上方伸出）
  - 整体线条硬朗，呈箭头/矛头形态
- **大小**：约 24px 机头到尾翼，翼展约 20px（随透视距离缩放）
- **动画**：左右移动时机身会横向倾斜（roll），类似战机侧身转向
- **操作**：W/A/S/D 上下左右移动（在第一章中受透视地面限制），空格键射击

### 第二章造型（侧面横版视角）
- **颜色**：同上，纯白线条
- **形状**：第一章的飞机旋转了 90°——机头朝右，尾翼朝左
- **操作**：W/A/S/D 上下左右自由移动（在廊桥内部），空格键射击

## 三、第一章：FIRST FLIGHT

### 背景元素
- **天空**：深灰黑色 (50,50,50)，布满 100 颗闪烁星星
- **地平线**：位于 y=200 处
- **地面**：灰色 (80,80,80)，从地平线向下延伸
- **透视线**：
  - 17 条放射线从屏幕底部向地平线上的消失点汇聚
  - 16 条深度横线，越靠近地平线越密，模拟纵深网格
  - 线条颜色：深灰 (45~50)

### 基地（地面核心）
- **形状**：位于屏幕底部中央的半个椭圆体（横轴半径 ~385px，纵轴半径 75px）
- **颜色**：
  - 外壳：翠绿色 (50, 180, 80)
  - 内部支柱：深绿色 (30, 130, 55)，30条竖直加强筋
  - 底部基线：翠绿色 (50, 180, 80)
- **含义**：这是玩家需要保卫的太空基地 / 母舰

### 外星敌机
- **基本造型**：菱形/钻石形（4顶点线框），上下左右四角对称
- **颜色**（动态）：
  - 满血时：偏红色 (255, 150, 100) — 接近橙红
  - 残血时：偏暗红 (255, 80, 30) — 更暗沉
  - 入场/无敌时：冰蓝色 (100, 180, 255) — 半透明发光感
- **入场特效**：身后拖出 3 个逐渐变淡的蓝色残影点
- **大小**：随距离动态缩放（远处小至 3px，近处大至 28px）
- **3 种入场方式**：
  1. 从画面左侧或右侧横向滑入
  2. 从画面顶部俯冲下垂
  3. 从 Boss 身上分裂飞出（Boss 战中）

### 子弹
- **玩家子弹**：白色细线，尾部有暗淡拖尾
  - 子弹能量随飞行距离衰减（约 370px 最大射程）
  - 衰减到末段时变为灰色，且失去伤害判定
- **Boss 吸收光束**：蓝色粗线 (100, 200, 255) / (60, 160, 255)，追踪目标敌机

### 冲击波（Shockwave）
- **形状**：从基地表面向上扩散的半椭圆环（与基地形状匹配）
- **颜色**：翠绿色 (50, 255, 100)，透明度随扩散距离迅速降低
- **触发条件**：每积攒 3 击杀 = 1 分，每 30 分升级一次冲击波（威力+1）
- **视觉**：冲击波从基地边缘呈椭圆状扩散上升，越远越透明
- **产生时**：基地表面飞出大量绿色粒子

### 粒子系统
- **爆炸粒子**（橙黄色）：敌机被消灭 / 碰撞时产生，向外飞散然后减速消失
- **绿色粒子**（冲击波产生时）：基地表面飞出
- **白色粒子**（子弹命中 / 其他）：小型高亮粒子
- **红色粒子**（Boss 吸能时）：从敌机飞向 Boss 血条的粒子流
- **烟花粒子**（通关后）：白色/绿色/红色混合，从屏幕中央向上喷发

## 四、第二章：DEEP SPACE（横版廊桥）

> 这是一个完全不同的空间场景。第一章是"在星球表面保卫基地"，第二章是"战机在太空基地之间的连接廊桥内部飞行"。

### 世界观场景
- 一条无限延伸的未来风透明廊桥，连接两个太空基地
- 战机在廊桥**内部**飞行，摄像机从侧面观看这个廊桥
- 廊桥本身快速向左滚动，表现出战机向前飞行的速度感

### 星空（保留元素）
- 400 颗闪烁星星布满全屏（70% 亮度变化范围）
- 10% 的星星有十字芒（2~4px 十字光）
- 星星以极慢速度（0.015x 视差）向左移动，几乎是静止的深空背景

### 金属地面
- **颜色**：深灰蓝色基底 (22, 24, 30)，填满画面下半部分（y=280 以下）
- **地面接缝**（5 条深度轨道）：
  - 横向短虚线，代表金属板的接缝
  - 越靠近"摄像机"（画面下方）的轨道，接缝越宽、越亮
  - 颜色：中灰 (45, 48, 58)，近处 alpha 180，远处 alpha 92
- **地面与玻璃墙交界线**：y=280 处，深灰 (35,38,45) + 略亮的细线 (55,58,68)

### 玻璃面板
- **风格**：Minecraft 风格的半透明玻璃
- **颜色方案**：
  - 外边框：淡蓝白 (170, 200, 230)，alpha 95
  - 内部网格线：淡蓝白 (165, 195, 225)，alpha 55（3 条横线 + 2 条竖线）
  - 四角高光：亮蓝白 (220, 235, 255)，alpha 130（Minecraft 玻璃标志性的角亮）
  - 中央光晕：浅蓝 (200, 225, 250)，alpha 45（斜向对角线微光）
- **形状**：每两根支柱之间填充一块梯形玻璃面板（上窄下宽，符合透视）
- **效果**：星空透过玻璃可见，带轻微淡蓝光泽

### 金属支柱
- **位置**：廊桥远端（画面深处），沿水平方向等距排列，间距约 330 世界单位
- **透视**：支柱向上延伸时略微倾向于消失点（位于画面中心上方 -60px 处）
- **颜色**：
  - 主体：深灰 (55, 55, 65)，alpha 230
  - 左侧边（亮面）：淡蓝灰 (80, 90, 120)，alpha 180
  - 右侧边（更亮面）：淡蓝灰 (110, 120, 145)，alpha 180
- **主支柱**：每 5 根有一根更粗的"主支柱"（5px 宽 vs 普通 3px）
- **底部三角支撑**（gusset）：支柱与地面交接处有小型三角加强结构，颜色 (65,65,75)
- **滚动**：所有支柱以 3.5 px/帧的速度向左滚动

### 透视设计
- **玻璃墙/支柱消失点**：VP = (CENTER_X=400, -60)，在画面之外的上方
- **地面接缝**：5 条固定 y 坐标的深度轨道（而非汇聚到消失点），从右往左水平移动
- **地面与玻璃墙夹角**：约 120°，两者在不同的平面上

## 五、Boss 系统

### Boss 造型
- **基本形状**：大型菱形/楔形（与敌机相似但更大）
  - 比普通敌机更扁：上方尖锐，下方稍平
  - 顶部到底部有中轴线
- **颜色**：
  - 主体：深红色 (200, 50, 50)
  - 外围光环：两层六边形环（浅红色 255, 100, 100），alpha 100
  - 受击闪白：(255, 200, 200)，持续 5~8 帧
- **大小**：80~100px（可配置，随章节递增），入场时从小变大
- **移动**：在画面中上部做正弦波往复运动（横向振幅 140~180px，纵向 25~45px）

### Boss 机制
1. **相位 1（正常战斗）**：
   - Boss 从屏幕上方降入（带缓动动画），着陆时产生屏幕震动 + 扩散红色圆形冲击波
   - 周期性发射**吸收光束**，抓取一只敌机螺旋吸入体内，增加自身 **Bonus HP**（粉色护盾）
   - 周期性释放**治疗波纹**（红色圆圈扩散），击中地图中的敌机后为其回血并增加 Boss 的 Bonus HP
   
2. **相位 2（半血触发）**：
   - Boss 再次升空，吸收场上剩余敌机，恢复部分 HP
   - 屏幕再次震动
   - 治疗波纹功能关闭
   - Boss 进入狂暴状态

3. **击败**：
   - Boss 被击倒后进入击败动画（300 帧爆炸粒子 → Boss 消失 → 逐一清除场上敌机）
   - "MISSION COMPLETE" 金色文字出现
   - 玩家自动返回画面中央
   - 战机向上飞出画面
   - 烟花粒子在画面中央爆发
   - 屏幕渐黑 → "MISSION COMPLETE" + "EXIT" 提示

### Boss HP Bar 设计
- 屏幕顶部居中，宽 400px，高 14px
- 底色：深灰 (30, 30, 30)
- 基础血量：深红色 (220, 30, 30)
- Bonus HP：粉色 (255, 140, 140)，叠在红色右侧
- 边框：浅灰色 (200, 200, 200)

## 六、HUD 与 UI

### 游戏内 HUD（右上角）
- **SCORE 分数**：白色文字 "SCORE N"
- **HP 血量**：10 个红色星号 `*`，每颗代表 1 点基地血量（被敌机撞击-1，归零=Game Over）
- **能量条**：分数进度条，填满后冲击波升级（0→30→60→90→120→150→180→MAX）
  - 未满时：绿色渐亮（lv1深绿 → lv6亮绿）
  - 满级时：整条亮绿

### 菜单系统
- **开始画面**：标题 "STAR FOX" + 副标题 "SPACE SHOOTER"，下方菜单选项（PLAY/CHAPTER/TEST/OPTIONS/EXIT）
- **章节选择**：5 个章节选项，未解锁的显示为灰色 + "(LOCKED)" 标记
- **暂停菜单**：半透明遮罩 + Resume/Restart/Options/Back to Menu/Exit
- **Game Over**：半透明遮罩 + "GAME OVER" + 分数 + "PLAY AGAIN"/"EXIT"
- **Mission Complete**：屏幕渐黑 → "MISSION COMPLETE" 大字 + "EXIT"

### 辅助瞄准系统
- 从玩家飞机位置延伸出一条瞄准线
- 自动吸附到距离最近的敌机（白色方框标记）
- 未吸附时显示为十字准星

## 七、程序音频设计

### BGM 系统
- **普通 BGM**：32 音符旋律循环，55Hz 低频贝斯脉冲，温和的合成器音色（正弦波+脉宽调制）
- **Boss BGM**：16 音符旋律循环（更高更紧张），65Hz 低频贝斯，更快的脉冲速度
- **波形**：正弦波（主旋律）+ 正弦波（贝斯）
- **音量**：旋律 0.045~0.055，贝斯 0.030~0.038

### 音效列表
| 音效 | 描述 | 音色 |
|------|------|------|
| 射击 | 高音快速下滑 | 正弦波扫频 1200→600Hz |
| 命中 | 中音短促 | 方波 350Hz |
| 小爆炸 | 噪声短爆 | 噪声 80Hz, 80ms |
| 大爆炸 | 噪声长爆 | 噪声 50Hz, 180ms |
| 冲击波 | 低沉上升 | 正弦波扫频 60→150Hz, 250ms |
| 冲击波命中 | 清脆击打 | 正弦波 100Hz, 60ms |
| Boss 治疗 | 柔和上扬 | 正弦波扫频 300→600Hz |
| Boss 吸收 | 低沉有力 | 正弦波扫频 200→80Hz |
| Boss 入场 | 多层下行音 | 5 个叠加音色（正弦+方波+噪声）, 低频到高频 |
| Boss 受击 | 刺耳金属感 | 方波 280Hz + 噪声 80Hz |
| 基地受损 | 低频轰击 | 噪声 45Hz |
| 屏幕震动 | 低频短噪声 | 噪声 35Hz |

## 八、游戏流程

1. **标题画面** → 选择 PLAY（快速开始当前章节）/ CHAPTER（选择章节）/ TEST（调试模式）
2. **选章** → 仅已解锁的章节可选（通关当前章节自动解锁下一章）
3. **倒计时** → 3-2-1 数字碎片动画（数字由粒子炸开）
4. **正常战斗** → 敌机从各方入场飞向基地，玩家用子弹和冲击波消灭之
5. **Boss 触发** → 分数达到阈值，Boss 从屏幕顶降入
6. **Boss 战** → 攻击 Boss 本体，同时应对其召唤的敌机、吸收光束、治疗波纹
7. **Boss 击败** → 击败动画 → 通关画面 → 解锁下一章 → 返回标题

## 九、视觉风格总结

- **核心美学**：极简镂空线条 + 矢量图形 + 深空黑背景
- **无填充色块**：所有实体均为线条勾勒（飞机、敌机、Boss、基地）
- **色彩基调**：
  - 第一章：灰+白+绿（基地绿是唯一鲜艳色调）
  - 第二章：深蓝灰+淡蓝玻璃+金属暗色（更冷峻的未来感）
- **动态元素**：闪烁星星、粒子爆炸、透视滚动、屏幕震动、Boss 光环
- **氛围方向**：复古街机矢量风格 × 太空科幻 × 抽象几何美学

> **为剧情编写提供的参考方向**：
> - 主角是一名星际飞行员，驻守在某行星表面的母基地
> - 第一章：外星舰队首次入侵，保护母基地
> - 第二章：追入太空，穿过连接轨道站的廊桥追击敌军
> - 第三至五章：深入敌军腹地，逐步揭开外星力量的真面目
> - 整体氛围：孤独的飞行员 vs 无尽的外星舰队，宏大而冷峻的太空歌剧
